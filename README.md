# WooCommerce Add Settings Example #

This is a sample plugin for WooCommerce.

## Description ##

When you enable this plugin, some setting fields can be added on the below WooCommerce settings tab.

* WooCommerce > Settings > Advanced

Note: You can also add settings different tabs, by adjusting tab name.

The fields to be added are as follows.

* Checkbox
* Radio
* Text
* Textarea
* Number
* Date

You can get the value of their fields as follows.

* `get_option( 'wcse_check' )`
* `get_option( 'wcse_radio' )`
* `get_option( 'wcse_text' )`
* `get_option( 'wcse_textarea' )`
* `get_option( 'wcse_number' )`
* `get_option( 'wcse_date' )`
